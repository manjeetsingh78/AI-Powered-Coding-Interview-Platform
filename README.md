# AI-Powered Coding Interview Platform

An AI-powered coding interview platform with three clear product roles:

- `candidate`: the regular user who practices problems, takes tests, and reviews feedback
- `interviewer`: the interviewer or examiner who creates assessments, evaluates candidates, and manages interviews
- `admin`: the full-control role that manages users, problems, permissions, and advanced platform operations

## Current foundation

- Frontend is now planned and scaffolded as `React + Vite`, not Next.js.
- Backend architecture is documented as `Django + DRF + Channels + Celery`.
- A feature-branch workflow is documented in `docs/BRANCHING_STRATEGY.md`.

## Repository structure

- `frontend/`: React frontend foundation and architecture guide
- `backend/`: Django architecture guide
- `docs/`: project planning and workflow docs
